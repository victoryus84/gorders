# GOrders Ecosystem

A professional, scalable solution for order and contract management, featuring a high-performance backend and a modern mobile client.

## 🏗 System Architecture
- **server:** Go (Gin, GORM, PostgreSQL) - Focused on Clean Architecture and ERP integration.
- **client:** Flutter - Cross-platform mobile application for real-time order tracking.
- **Infrastructure:** Docker & Docker Compose for seamless deployment.

1. [Backend Module](./server) - Dive into the API documentation and database schema.
2. [Frontend Module](./client) - Explore the mobile UI and state management logic.
